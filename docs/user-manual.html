系统概述

兔子进销存是一款轻量级移动进销存管理系统，专为个体商户和小型批发商设计。支持手机浏览器和微信小程序两种方式使用，无需安装 APP，打开即可操作。

核心功能

模块功能
📊 首页经营概览、KPI 统计
🧾 开单销售下单、收款
📦 进货采购入库、进价记录
📋 库存商品管理、库存预警
👤 我的账户管理、客户/进货商

适用场景

  • 建材五金、汽配摩配
  • 酒水饮料、副食干杂批发
  • 藏药材、特产批发
  • 农资饲料、小型加工作坊
  • 日均开单 5-30 笔的商户
💡 提示：本系统适合"低频但有记账需求"的商家。高频零售（便利店/超市）建议使用专业 POS 系统。

常见问题

Q1: 试用到期后怎么办？

联系客服购买正式套餐。到期前系统会在首页顶部显示剩余天数，到期后系统锁定，只能看到续费联系页面。

Q2: 如何修改商品信息？

库存管理 → 点击商品卡片 → 进入编辑页 → 修改字段 → 保存。

Q3: 开单时搜不到商品？

请先在「库存管理 → 新增商品」中创建商品。商品必须提前录入系统才能开单。

Q4: 进货时找不到进货商？

请先去「我的 → 进货商管理」添加进货商。进货必须先有进货商。

Q5: 扫码功能怎么用？

开单页点击 📷 按钮，对准商品条码扫描。系统自动搜索匹配商品并添加到购物车。

Q6: 如何退款？

首页 → 今日销售 → 点击订单进入详情 → 点击底部"退款"按钮。退款后库存自动恢复。

Q7: 数据安全吗？

所有数据加密传输（HTTPS），数据库每日备份。服务器位于腾讯云，提供 99.95% 可用性保障。

Q8: 可以多人同时使用吗？

当前版本每个账号支持单设备登录。如需多店员同时操作，可联系客服了解多用户版本。

Q9: 忘记密码怎么办？

目前请联系客服重置密码。后续版本将支持短信验证码重置。

Q10: 支持导出数据吗？

当前版本暂不支持批量导出。如需导出销售报表或库存清单，请联系客服协助处理。
